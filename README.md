Proyecto  Fashion Insights
___
Este repositorio contiene un proyecto desarrollado en Tableau compuesto por 3 dashboards interactivos enfocados en el análisis de ventas, clientes y métricas de negocio.

📌 Resumen

Dashboard general con los principales indicadores del negocio:

Ventas totales
Beneficio generado
Cantidad de pedidos
Tendencias generales y métricas clave

👥 Clientes

Dashboard orientado al análisis de clientes:

Segmentación de clientes
Comportamiento de compra
Distribución geográfica
Métricas de clientes recurrentes

📈 Análisis

Dashboard enfocado en análisis detallado de rendimiento:

Comparación de categorías y productos
Análisis de ganancias y pérdidas
Tendencias de ventas
Insights para la toma de decisiones

🎯 Objetivo del proyecto

Transformar datos en visualizaciones interactivas para facilitar el análisis empresarial y apoyar la toma de decisiones basada en datos.
